WHAT THIS IS
------------
This is the merge you asked for: the best copy of every piece of
VERITAS and Blackletter, gathered into one place, so nothing more
gets lost. Every file in here was checked before it went in. Where
two copies of the same thing existed, they were compared and only
the better one was kept. The full list of what won, and why, is in
MASTER_MANIFEST.txt.

From today forward, THIS folder is the one true copy. If a file is
not in here, treat it as retired.

WHAT IS IN EACH NUMBERED FOLDER
-------------------------------
  01_APP              The VERITAS desktop application. This is the
                      newest version of every code file. All 401
                      self-checks pass.

  02_GATE_BLACKLETTER The three working Blackletter programs: the
                      gate, the engine, and Nova Johnson. All three
                      were run and watched working before packing.

  03_LIBRARIES        The four knowledge layers, one canonical copy
                      of each:
                        - definitions library (209 entries)
                        - etymology dictionary (876 entries)
                        - historical context library (11 sources)
                        - usage database (1,397 words, 4,145 quotes)

  04_WORD_RECORDS     The word-record app (1,396 words, 9 columns),
                      Johnson's Bench (1,334 verified), and the
                      usage viewer. Open these by double-clicking;
                      they run in your web browser.

  05_AUDIT            The full definition audit. Its numbers were
                      independently recounted and they are honest.

  06_SPEC_STONES      The written rules of the system. The rules
                      come before the code; these are the rules.

  07_CAPTURE_TOOLS    The two tools that pull clean, human-typed
                      Johnson text from johnsonsdictionaryonline.com
                      one word at a time. This is the path that no
                      bulk-download block can stop.

  08_RAW_QUARANTINE   Unverified raw material. NOTHING in this
                      folder is allowed to feed a decision until a
                      human has verified it. It is kept so no work
                      is lost, not because it is trusted.

  09_RECORDS          The paper trail: which old files were judged,
                      which were kept, which were discarded, and why.

ONE THING KEPT SEPARATE
-----------------------
VERITAS_KEEP.zip (the big 107MB source vault with the OED volumes,
Blackstone, Farrand, Elliot, the Federalist, and the rest) is NOT
inside this package -- it is already a finished archive and copying
it in would only double its size on your machine. Keep that zip in
the same folder where you keep this package. Together, the two of
them are everything.

THE STATE OF THE DECIDER (so it is written down in one place)
-------------------------------------------------------------
The design is a two-out-of-three decider, growing to three-out-of-
three: etymology + definition + use at the time. As of this merge:

  Etymology layer:   876 words, verified, clean.        READY.
  Definition layer:  209 promoted; 1,334 more verified
                     Johnson texts waiting in the Bench. NEARLY READY.
  Use layer:         1,397 words with dated quotes, but
                     some quotes come from a modern
                     editor's introduction and must be
                     stripped before this layer decides.  NEEDS CLEANING.

The gate code currently lets only the definition layer decide. The
rewrite to the two-out-of-three rule is the next build step, after
this merge is safely on your machines.

KEEP IT SAFE
------------
Put a copy of this package on BOTH computers, and if you have a USB
stick, a third copy there. Three places, two machines, one truth.
