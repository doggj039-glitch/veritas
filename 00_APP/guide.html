VERITAS resolves the words of the Constitution to their founding-era meaning (Johnson's 1773 Dictionary), shows how each word drifted afterward, and lets you read the founding record itself. Everything runs on your machine.

01Quick start

1
Type a word in the Ask box (bottom-center) — try commerce — and press ASK.
2
You get the verbatim 1773 definition, its etymology, how the founders used it in the record, and the Supreme Court cases that later stretched it.
3
Try a word that isn't founding-era — interstate — and it returns Void for Vagueness instead of guessing.
4
Change the dropdown next to the Ask box from Define · gate to a voice (say Blackstone) and ask a question — you get his actual words, cited.
5
Open the ☰ menu (top-left) to switch windows, or come back to this guide anytime.

02The Ask box & its three modes

The dropdown beside the Ask box sets what your question does:

ModeWhat it does
Define · gateResolves a word to its verbatim Johnson 1773 meaning, or flags it Void for Vagueness.
Ask a voicePuts your question to a grounded agent — Publius, Blackstone, Montesquieu, Story, the Convention, Rawle, Lieber, Sedgwick, Anti-Federalist, Ratification. Verbatim, cited answers only.
Discuss a topicRuns a citizen/Socratic walkthrough of an analysis dataset — premise, grounded questions, conclusion.
Ask Claude · groundedNeeds an API key (Settings) + internet. Retrieves the grounded sources locally first, then a Claude model composes a cited answer from only those. Shows the terms it grounded on and the tokens used.

Everything you ask lands in the workspace and stays for the session. The Clear session button empties it.

03Reading a result card

When you define a word, the card is layered top-to-bottom:

  • Verdict — ADMITTED (grounded) or VOID FOR VAGUENESS (no founding authority).
  • Johnson 1773 · definition — the sense, verbatim. (Johnson's literary example-quotations are omitted for clarity.)
  • Etymology and Founding-era usage — real quotations from the founding documents, cited.
  • Construed by the Supreme Court — the cases that moved the word, with the direction of drift, and the dissents that argued against it.
Read a "Void" correctly

Void for Vagueness means "not a Johnson 1773 headword" — a tripwire for undefined operative vocabulary. It is not a ruling that something is unconstitutional.

04Reading the books

Open the Library tree on the left (Primary window). Under Founding-Era Sources click a book — Blackstone, Montesquieu, the Federalist, Elliot's Debates, and more. Its card has a "Read the full text →" button that opens the whole book in a reader you can scroll, right inside the app. Close it with the ✕ or the Esc key.

Ten founding-era works are fully readable this way — every word verbatim, in document order.

05Browsing the Supreme Court cases

Open Case Detail from the ☰ menu. The left column starts with the landmark cases (each with a full founding-grounding analysis and its dissent). Type in the search box and you search the entire corpus — 7,696 cases (1956–2018). Click any result to see its verified record and links to the full opinion on Oyez and Justia.

06The windows (the ☰ menu)

The three-line icon, top-left, opens the window switcher. VERITAS is a hub with spokes:

Use the ◀ Back / ▶ Forward arrows (top-left) to retrace your steps between windows. The Research Panel on the right collapses with its ‹ button and reopens from the tab on the right edge.

07Settings & the API key

The ⚙ gear (top-right) shows the engine status and lets you add an Anthropic API key to enable the grounded "Ask Claude" mode. The key is stored only on your machine (~/.config/veritas, permissions 600) and is never placed in the page. VERITAS works fully without it.

08Offline now, online later

Today VERITAS is fully offline — the gate, the ten agents, the dictionaries, the corpora, and the case index all run locally with no internet. The optional online tier (with an API key) now lets a Claude model compose over these grounded sources — retrieving the verbatim passages and citing them, with the gate fencing every term. It never overrides the founding meaning; it only arranges what the sources say.

09How it stays honest

  • Verbatim only — never invents or paraphrases a founding-era meaning.
  • 1773 is controlling — etymology, usage, and drift corroborate; later dictionaries never override.
  • When ungrounded, it says so — Void for Vagueness beats a confident guess.
  • Analysis is labeled as analysis — the Discuss walkthroughs are arguments to check against the gate, never verdicts.

If anything here stops matching what the app does, the app is right and the manual is stale — tell whoever maintains it to fix the manual first.
